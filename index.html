Integration by Parts

Substitution Rule

  • u-substitution: ∫ f(g(x)) g'(x) dx = ∫ f(u) du
  • Standard substitutions for composite functions
  • Rational substitution (z2 = linear expression)
  • ∫ x2 sin x3 dx, ∫ (1−1/x2) ex+1/x dx

Trigonometric Substitution

  • x = a sin θ for √(a2 − x2)
  • x = a tan θ for √(a2 + x2)
  • x = a sec θ for √(x2 − a2)
  • Worked: ∫ dx/(a2+x2)2

Standard Integrals

  • ∫ 1/(a2+x2) dx = (1/a) tan−1(x/a) + C
  • ∫ 1/(a2−x2) dx = (1/2a) log|(a+x)/(a−x)| + C
  • ∫ 1/√(a2−x2) dx = sin−1(x/a) + C
  • ∫ 1/√(x2+a2) dx = log(x+√(x2+a2)) + C
  • ∫ √(x2±a2) dx forms

Integrals Reducible to Standard Form

  • ∫ 1/(ax2+bx+c) dx — complete the square
  • ∫ (mx+e)/(ax2+bx+c) dx — split into p(2ax+b)+q
  • ∫ 1/√(ax2+bx+c) dx
  • ∫ (mx+e)/√(ax2+bx+c) dx

Partial Fractions

  • Linear factors — (x−r)m decomposition
  • Quadratic factors — (x2+px+q)n decomposition
  • Long division when degree of numerator ≥ denominator
  • Solving for constants by substitution or equating coefficients

Improper Integrals

  • Type I — infinite limits of integration
  • Type II — unbounded integrand on [a, b]
  • Convergence: limit is finite
  • Divergence: limit is ±∞ or does not exist
  • Splitting at a point for doubly-infinite integrals

Applications of Definite Integrals

  • Area between two curves
  • Volume by Disk method (rotation about x- or y-axis)
  • Volume by Washer method
  • Arc length of plane curves (Cartesian & parametric)

✏️ Worked Examples

Five of the hardest problems from your notes, solved step by step.

EX 1

FTC + Chain Rule — Find dy/dx if y = ∫1x² cos t dt

1
Identify the structure
The upper limit is x2, not x. This means y is a composite function. Let u = x2, so y = ∫1u cos t dt.
2
Apply the Chain Rule
dy/dx = (dy/du) · (du/dx). By FTC Part I, dy/du = cos u.
dy/dx = (d/du ∫1u cos t dt) · (du/dx) = cos u · du/dx
3
Substitute back u = x²
du/dx = 2x, and cos u = cos(x2).
dy/dx = cos(x2) · 2x = 2x cos(x2)
ANSWERdy/dx = 2x cos(x2)
EX 2

Integration by Parts — Evaluate ∫ ln x dx

1
Choose u and dv
Write ∫ ln x · 1 dx. By the LIATE rule, choose u = ln x (logarithm) and dv = dx.
u = ln x   →   du = (1/x) dx
dv = dx   →   v = x
2
Apply the formula ∫ u dv = uv − ∫ v du
∫ ln x dx = (ln x)(x) − ∫ x · (1/x) dx = x ln x − ∫ 1 dx
3
Evaluate the remaining integral
= x ln x − x + C
ANSWER∫ ln x dx = x ln x − x + C
EX 3

Trig Substitution — Evaluate ∫ dx / (a2+x2)2

1
Choose the substitution
The form a2+x2 calls for x = a tan θ, dx = a sec2θ dθ.
a2 + x2 = a2 + a2 tan2θ = a2(1 + tan2θ) = a2 sec2θ
2
Substitute and simplify
∫ dx/(a2+x2)2 = ∫ [a sec2θ dθ] / [a4 sec4θ] = (1/a3) ∫ cos2θ dθ
3
Use the half-angle identity: cos2θ = (1 + cos 2θ)/2
= (1/2a3) ∫ (1 + cos 2θ) dθ = (1/2a3) [θ + (1/2) sin 2θ] + C
4
Convert back to x
Since x = a tan θ, we have θ = tan−1(x/a). Also sin 2θ = 2 sin θ cos θ. From the right triangle: sin θ = x/√(a2+x2), cos θ = a/√(a2+x2).
(1/2) sin 2θ = sin θ cos θ = x/√(a2+x2) · a/√(a2+x2) = ax/(a2+x2)
ANSWER(1/2a3) [tan−1(x/a) + ax/(a2+x2)] + C
EX 4

Partial Fractions — Evaluate ∫ (5x−3) / [(x+1)(x−3)] dx

1
Write the partial fraction decomposition
(5x−3) / [(x+1)(x−3)] = A/(x+1) + B/(x−3)
2
Clear denominators
5x − 3 = A(x−3) + B(x+1)
3
Find A and B by substitution
Put x = 3: 15−3 = 4B → B = 3. Put x = −1: −5−3 = −4A → A = 2.
A = 2,   B = 3
4
Integrate term by term
I = 2 ∫ 1/(x+1) dx + 3 ∫ 1/(x−3) dx = 2 ln|x+1| + 3 ln|x−3| + C
ANSWER2 ln|x+1| + 3 ln|x−3| + C
EX 5

Washer Method — Volume: y = x2+1 and y = −x+3 rotated about x-axis

1
Find intersection points
Set x2+1 = 3−x → x2+x−2 = 0 → (x+2)(x−1) = 0.
x = −2 and x = 1
2
Identify outer and inner radii
On [−2, 1], check which function is on top: at x = 0, y = 1 for the parabola and y = 3 for the line. So R(x) = −x+3 (outer), r(x) = x2+1 (inner).
3
Apply Washer formula V = ∫ π{[R(x)]² − [r(x)]²} dx
V = π ∫−21 [(−x+3)2 − (x2+1)2] dx
4
Expand and integrate
Expand the squares, combine like terms, then integrate. From your notes the answer is:
V = 117π/5
ANSWERV = 117π/5

⚠️ Common Pitfalls & Exam Tips

The most frequent errors students make — and how to avoid them on test day.

⛔ Common Pitfalls

⛔ Forgetting the Chain Rule with FTC

When the upper limit is g(x) (not just x), you must multiply by g'(x). Writing d/dx ∫ax² f(t) dt = f(x2) misses the factor of 2x.

⛔ Wrong sign when limits are flipped

d/dx ∫x5 f(t) dt = −f(x). Flipping the limits introduces a negative sign. Missing this is a very common error.

⛔ Bad choice of u in IBP

Always use LIATE: Logarithms > Inverse trig > Algebraic > Trig > Exponential. Choosing u = ex when x is present usually makes the integral harder, not easier.

⛔ Forgetting + C for indefinite integrals

Every indefinite integral needs the constant + C. Losing a mark here is avoidable — make it a habit to write it every single time.

⛔ Skipping long division in partial fractions

If deg(numerator) ≥ deg(denominator), you must do polynomial division first before decomposing. Decomposing directly gives wrong results.

⛔ Not converting back from θ to x in trig sub

After integrating in θ, always draw the reference right triangle and express sin θ, cos θ, tan θ in terms of x. Leaving the answer in θ is incomplete.

⛔ Assuming an improper integral converges

Always take the limit explicitly. ∫1∞ 1/x dx diverges (limit → ∞) even though 1/x → 0. Never assume a "shrinking" integrand gives a converging integral.

⛔ Washer vs Disk confusion

If the region does NOT touch the axis of rotation, use Washers. If it does touch, use Disks. Confusing these gives π R2 instead of π(R2 − r2).

✅ Last-Minute Exam Tips

✅ Memorise the 6 standard integral formulas

The arctan, arcsin, and log forms appear in almost every exam. Write them on your rough paper at the start of the test.

✅ For any rational integral, check degree first

Take 5 seconds at the start: is deg(top) ≥ deg(bottom)? Yes → long divide first. This single habit prevents a whole class of errors.

✅ Complete the square before integrating

For ax2+bx+c in the denominator, always complete the square to write it as (x+h)2 + k2. This unlocks the arctan standard form.

✅ Draw a right triangle for trig substitution

Label the sides from your substitution (e.g. x = a tan θ → opposite=x, adjacent=a, hypotenuse=√(a2+x2)). This makes back-substitution mechanical and error-free.

✅ Check volume by symmetry when possible

If the region is symmetric about the axis of rotation, integrate over half and double. This halves the work and provides a built-in check.

✅ Verify IBP by differentiating your answer

If time allows, differentiate your answer. Since d/dx[F(x)] = f(x), this is the fastest way to catch arithmetic errors without starting over.
